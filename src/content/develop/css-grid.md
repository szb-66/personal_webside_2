# CSS Grid 布局指南

## 什么是 Grid

CSS Grid 是一个二维布局系统。

## 基本语法

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

## 常用属性

- grid-template-columns
- grid-template-rows
- gap
- grid-area

## 总结

Grid 让复杂的布局变得简单。
